# 3. 初次使用

## 启动 CLI

安装完成后，直接在终端运行：
```bash
hermes
```

你会看到欢迎界面，包含以下信息：
- 当前激活的模型
- 可用工具数量
- 可用技能数量
- 常用命令提示

## 第一次对话

在提示符后输入你的问题，比如：
```
> 帮我写一个 Python 快速排序的实现
```

Hermes Agent 会开始思考，你会看到可爱的动画 spinner，然后返回结果。

## 常用基础命令

### 查看帮助
```
/help
```
显示所有可用的斜杠命令和使用说明。

### 切换模型
```
/model anthropic/claude-3-5-sonnet-20240620
```
切换到指定的模型。

### 切换皮肤
```
/skin ares
```
切换到指定的主题皮肤，内置皮肤包括 default、ares、mono、slate。

### 查看会话历史
```
/history
```
查看当前会话的所有历史消息。

### 清空会话
```
/clear
```
清空当前会话的上下文，开始新的对话。

### 退出 CLI
```
/exit
```
或者使用快捷键 `Ctrl+D`。

## 第一个工具调用示例

尝试让 Hermes Agent 列出当前目录的文件：
```
> 列出当前目录下的所有文件
```

如果工具权限正常，Hermes 会自动调用 `list_dir` 工具，返回当前目录的文件列表。

## 第一个技能使用示例

如果安装了 `alpha-loop` 技能，可以尝试：
```
> 用alpha-loop完成一个 Todo 列表应用的开发
```
技能会自动驱动迭代，直到完成任务。
