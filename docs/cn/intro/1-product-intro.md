# 1. 产品介绍

## 什么是 Hermes Agent

Hermes Agent 是一个开源的多功能 AI 智能体框架，支持多平台部署、工具调用、技能扩展，是为开发者和企业打造的通用 AI 助手解决方案。

## 核心功能特点

### 1. 多平台支持
- 本地 CLI 交互：支持终端下交互式使用
- 消息网关：支持 Telegram、Discord、Slack、WhatsApp、HomeAssistant、Signal 等主流消息平台
- IDE 集成：支持 VS Code、Zed、JetBrains 等 IDE 的 ACP（编辑器适配协议）适配器

### 2. 强大的工具系统
- 内置数十种内置工具：文件操作、网络搜索、浏览器自动化、代码执行、子代理调度、MCP（模型上下文协议）客户端等
- 工具权限控制：支持按平台启用/禁用工具
- 危险操作审批机制：自动检测高危命令并请求用户确认

### 3. 灵活的技能系统
- 技能即插即用：支持动态加载技能扩展功能
- 技能市场：内置技能市场，支持搜索、安装、管理技能
- 自定义技能开发：支持用户开发自定义技能满足个性化需求

### 4. 高性能优化
- 上下文自动压缩：智能压缩上下文，降低 Token（令牌）消耗
- 提示词缓存：支持 Anthropic 提示词缓存，大幅降低成本和响应时间
- 多模型支持：支持 Anthropic、OpenAI 等主流大语言模型（LLM）

### 5. 多配置集（Profile）隔离
- 支持多 Profile 配置：不同配置集完全隔离，互不影响
- 独立的配置、密钥、会话存储：每个 Profile 拥有独立的运行环境

### 6. 会话管理
- SQLite 会话存储：支持 FTS5 全文搜索
- 会话历史持久化：所有对话历史自动保存
- 会话轨迹导出：支持导出对话轨迹用于调试和分析

## 适用场景

1. **个人开发者助手**：本地 CLI 交互，辅助开发、调试、查文档
2. **团队协作助手**：部署到消息平台，作为团队内部 AI 助手
3. **企业级服务**：作为企业内部 AI 服务，集成到内部系统
4. **自定义 AI 应用**：基于 Hermes Agent 框架二次开发，打造专属 AI 应用
