# 4. 技能系统使用

Hermes Agent 的技能系统是扩展功能的核心机制，通过安装不同的技能，可以让 Agent 获得特定领域的专业能力，满足个性化和场景化需求。

## 概述

技能是一种可插拔的功能扩展包，每个技能专注于解决特定领域的问题，比如代码评审、数据分析、文档写作等。技能以目录形式存储，包含提示词逻辑、参考模板、工具配置等内容，无需修改核心代码即可动态加载使用。

### 技能加载机制
1. **存储结构**：每个技能是一个独立的目录，包含 `SKILL.md` 核心文件（YAML 前置信息 + 提示词逻辑），以及可选的参考文件、模板、脚本等资源。
2. **自动扫描**：启动时系统会自动扫描 `~/.hermes/skills/` 目录和自定义的外部技能目录，解析 `SKILL.md` 文件生成对应的 `/技能名` 命令。
3. **调用流程**：用户触发技能时，系统自动加载技能的提示词内容，注入配置变量，附加用户指令后发送给 Agent 执行。
4. **预加载支持**：启动时可以通过 `--skills [技能名1,技能名2]` 参数预加载技能，技能逻辑会在整个会话周期内自动生效。

## 技能管理

所有技能管理命令可以在 CLI 中直接使用：

### 查看已安装的技能
```
/skills list
```
列出所有已安装的技能，包含技能名称、版本、描述信息。

### 搜索技能市场
```
/skills search [关键词]
```
在官方技能市场中搜索相关技能，比如：
```
/skills search python
```

### 安装技能
```
/skills install [技能名称]
```
安装指定的技能，比如安装代码重构技能：
```
/skills install python-refactor
```

### 升级技能
```
/skills update [技能名称]
```
升级指定技能到最新版本，如果不指定技能名则升级所有已安装的技能。

### 卸载技能
```
/skills uninstall [技能名称]
```
卸载指定的技能。

### 启用/禁用技能
```
/enable skill [技能名称]   # 启用技能
/disable skill [技能名称]  # 禁用技能
```
禁用的技能不会被加载，也无法被调用。

## 技能使用方式

### 1. 显式触发（推荐）
用户主动输入 `/技能名 [用户指令]` 触发技能执行，这是最常用的方式：
```
# 触发代码评审技能，评审当前目录下的代码
/code_review 评审 src/ 目录下的所有 Python 代码，重点关注性能和安全问题

# 触发文档写作技能，生成项目 README
/cn-doc-writer 为当前项目生成中文 README 文档，包含功能介绍、安装步骤、使用说明
```

### 2. 隐式触发
预加载的技能会自动在会话中生效，不需要显式调用：
```bash
# 启动时预加载翻译技能
hermes --skills translation
```
启动后，所有对话都会自动应用翻译技能的逻辑，用户不需要每次都输入 `/translation` 命令。

### 3. 技能参数
部分技能支持传递参数，参数格式为 `--参数名 参数值`：
```
# 生成指定风格的产品原型
/product_design 生成电商APP的产品原型 --style 简约风 --output prototype.md
```

## 自定义技能加载

### 本地自定义技能
1. 在 `~/.hermes/skills/` 目录下创建新的技能目录，比如 `my_skill`
2. 在目录下创建 `SKILL.md` 文件，按照技能规范编写技能逻辑
3. 重启 Hermes Agent，系统会自动加载新技能

### 外部目录技能
可以通过配置 `HERMES_SKILL_PATHS` 环境变量添加自定义的技能目录：
```env
HERMES_SKILL_PATHS=/path/to/my/skills,/another/path
```
系统启动时会同时扫描这些目录下的技能。

## 常见问题

**Q：安装技能后不生效？**
A：重启 Hermes Agent 即可，技能会在启动时自动加载。

**Q：如何查看技能的使用说明？**
A：输入 `/技能名 --help` 即可查看该技能的详细使用说明和参数说明。

**Q：技能调用失败怎么办？**
A：检查技能是否已启用，是否缺少必要的依赖或配置，查看错误信息进行排查。

**Q：可以分享自定义技能给其他人吗？**
A：可以，将技能目录打包发送给其他人，放到对方的 `~/.hermes/skills/` 目录下即可使用。
