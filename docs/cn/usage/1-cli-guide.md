# 1. CLI 交互指南

## 交互界面说明

Hermes CLI 界面由以下部分组成：
1. **欢迎横幅**：启动时显示，包含版本、模型、工具、技能等信息
2. **输入提示符**：用户输入命令和问题的地方
3. **思考 spinner**：模型处理时显示的动画，包含当前动作和可爱的表情
4. **工具输出**：工具调用的结果，以特殊前缀标识
5. **响应框**：模型的最终响应，以边框包围显示

## 斜杠命令系统

所有斜杠命令以 `/` 开头，支持自动补全，按 `Tab` 键可以查看可用命令和参数。

### 会话类命令

| 命令 | 别名 | 说明 | 示例 |
|------|------|------|------|
| `/clear` | `/c` | 清空当前会话上下文 | `/clear` |
| `/history` | `/h` | 显示当前会话历史 | `/history` |
| `/save` |  | 保存当前会话到文件 | `/save my_session.json` |
| `/load` |  | 从文件加载会话 | `/load my_session.json` |
| `/exit` | `/quit`、`/q` | 退出 CLI | `/exit` |

### 配置类命令

| 命令 | 别名 | 说明 | 示例 |
|------|------|------|------|
| `/set` |  | 设置配置项 | `/set display.skin ares` |
| `/get` |  | 查看配置项 | `/get model.default` |
| `/model` | `/m` | 切换默认模型 | `/model openai/gpt-4o` |
| `/skin` |  | 切换主题皮肤 | `/skin mono` |
| `/setup` |  | 重新运行配置向导 | `/setup` |

### 工具与技能类命令

| 命令 | 别名 | 说明 | 示例 |
|------|------|------|------|
| `/tools` |  | 管理工具集 | `/tools list` |
| `/skills` |  | 管理技能 | `/skills install python-refactor` |
| `/enable` |  | 启用工具/技能 | `/enable tool browser` |
| `/disable` |  | 禁用工具/技能 | `/disable skill alpha-loop` |

### 信息类命令

| 命令 | 别名 | 说明 | 示例 |
|------|------|------|------|
| `/help` | `/?" | 显示帮助信息 | `/help model` |
| `/version` | `/v` | 显示版本信息 | `/version` |
| `/status` |  | 显示当前状态 | `/status` |
| `/models` |  | 列出可用模型 | `/models` |
| `/skins` |  | 列出可用皮肤 | `/skins` |

## 高级交互功能

### 多行输入
输入 `"""` 然后回车，进入多行输入模式，输入结束后再输入 `"""` 并回车发送。适合输入长文本或代码块。

### 命令补全
按 `Tab` 键自动补全命令、参数、模型名、皮肤名等，支持模糊匹配。

### 历史记录
按 `↑` 和 `↓` 键浏览历史输入，按 `Ctrl+R` 搜索历史输入。

### 输出控制
- 按 `Ctrl+C` 中断当前模型请求或工具执行
- 长输出会自动分页，按 `空格键` 翻页，按 `q` 退出分页

## 背景任务管理

### 运行后台任务
在命令末尾添加 `&` 或者使用 `background=true` 参数，让任务在后台运行：
```
> 运行 python app.py 后台执行 &
```
或者
```
> terminal(command="python app.py", background=true)
```

### 查看后台任务
```
/jobs
```
列出所有正在运行的后台任务。

### 查看任务输出
```
/jobs output 1
```
查看 ID 为 1 的后台任务的输出。

### 终止后台任务
```
/jobs kill 1
```
终止 ID 为 1 的后台任务。

### 等待任务完成
```
/jobs wait 1
```
等待 ID 为 1 的后台任务完成。
